# Plurk IMG viewer v0.3.0

> 最後更新日 2021/05/25

## 功能

- 快速檢視河道圖片
- 預覽範圍：
  1. 使用者上傳到河道的圖片
  2. twitter 貼文的附圖
  3. Youtube 影片的自訂縮圖
